<h4 id=西安_1>西安</h4>
方法一：以标题西安为例。标题加一个id等于目录上西安的锚点西安_1。这种方式锚点中可以有特殊字符。
``` markdown
<h4 id=西安_1>西安</h4>
```

#### 纽约
方法二：以纽约为例。markdown的标题会自动加上锚点。例如下面的锚点自动就是纽约。不过这种方式设置的锚点中不能设置特殊字符。打开开发者模式查看一下元素就明白了。
``` markdown
#### 纽约
```
